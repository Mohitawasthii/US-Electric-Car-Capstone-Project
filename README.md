## 📌 Project Overview

The **US Electric Car Capstone Project** is an end-to-end Data Analytics and Machine Learning project that analyzes electric vehicle (EV) registration data across the United States. The project explores EV adoption trends, manufacturer performance, model popularity, electric range, and geographic distribution while applying predictive analytics to generate data-driven insights.

## 🎯 Project Objectives

- Analyze EV registration and adoption trends.
- Perform data cleaning and preprocessing.
- Conduct Exploratory Data Analysis (EDA).
- Explore manufacturer and model performance.
- Study electric range and vehicle characteristics.
- Build predictive machine learning models.
- Generate actionable business insights.

## 📂 Dataset Features

The dataset includes information such as:

- VIN (Partial)

- State
- Postal Code
- Model Year

- Model
- Electric Vehicle Type
- Electric Range
- Base MSRP
- Legislative District
- Electric Utility
- CAFV Eligibility
